About VibeLog

VibeLog is a dark, minimalist blog built on raw C — no frameworks, no ORMs, no safety nets. Just pointers, file descriptors, and sheer engineering discipline.

Every page you see is served by a hand-crafted HTTP server compiled from pure C, reading articles from the filesystem like nature intended. No databases were harmed in the making of this blog.

The Stack

  • Language: C (ISO C11)
  • Server: CWebStudio — lightweight HTTP in C
  • Database: The filesystem itself — JSON files and directories
  • Frontend: Server-rendered HTML with inline CSS
  • Philosophy: Vibecoding — code as craft, minimalism as principle

Why?

Because sometimes the best framework is no framework. Because understanding every byte that leaves your server is a form of respect for the machine and the reader. Because we believe that constraints breed creativity, and C is the ultimate constraint.

Open Source

VibeLog is open source. The entire codebase — server, build system, and content pipeline — is available for inspection, forking, and judgment.
