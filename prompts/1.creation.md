### VibeLog


objetivo:
Criar um blog editando o **app.c** chamdo **mateusmoutinho** que deverá ter um aspécto moderno e minimalista porém sombrio e meio "demoníaco" com cores escuras.

passos:
1. Criar a rota main a rota **/** que deverá conter os seguintes intens:
 - uma navbar com os seguintes intens:
    - botão lua que redireciona para (**/list_article?page=1&category=lua&limit=10**)   
    - botão  C   (**/list_article?page=1&category=C&limit=10**)  
    - Botão Vibecoding (**/list_article?page=1&category=Vibecoding&limit=10**)  
    - Botão Software Architecture (**/list_article?page=1&category=Software Architecture&limit=10**)  
    - um input de busca que o item digitado (**/list_article?page=1&limit=10&search=<input do usuario>**)  
- Um Corpo Contendo os 5 artigos mais recentes obtidos no banco  de dados 


2. Criar a rota **/list_articles?page=1&category=lua&limit=5&search=lua-for-beguinners** que deverá retornar um html com a mesma navbar da rota **/** e o corpo contendo o titulo dos artigos, a data de criação,o resumo do artigo e a tumbnail dos artigos que correspondam a filtragem 


3. Criar a rota **/article?data=13-02-2026&id=id_do_artigo** que deverá retornar um html com a mesma navbar da rota **/** e o corpo contendo o titulo do artigo, o conteudo do artigo na linguagem do usuário (se houver tradução), caso contrário retorne o texto em inglês, e um footer contendo o autor, e a foto de perfil do autor.

4. Criar a rota **/asset?path=caminho_do_arquivo** que deverá retornar o arquivo solicitado.

5. Criar a rota **/author?id=autor_id**  que terá a  navbar padrão e retornará a foto de perfil do autor, e a descrição completa dele.

6. Criar a rota **/about** que retorá html de about 

especificação do banco de dados:

o banco de dados é uma pasta chamada **database** que seguirá o seguinte formato


~~~txt
/database
/articles
/articles/13-02-2026/
/articles/13-02-2026/<article_id>/data.json
/articles/13-02-2026/<article_id>/content/pt.html
/articles/13-02-2026/<article_id>/content/en.html
/articles/13-02-2026/<article_id>/assets/picture.jpg
/articles/13-02-2026/<article_id>/assets/thumbnail.jpg
/authors/<author_id>/data.json
/authors/<author_id>/assets/picture.jpg

~~~
- schema de :**/articles/13-02-2026/<article_id>/data.json**
~~~json
{
    "title": "titulo_do_artigo",
    "summary": "resumo_do_artigo",
    "author": "autor_do_artigo",
    "thumbnail": "caminho_da_thumbnail",
    "categorys": ["categoria1", "categoria2", "categoria3"]
}
~~~

- schema de :**/authors/<author_id>/data.json**
~~~json
{
    "name": "nome_do_autor",
    "description": "descricao_do_autor",
    "picture": "caminho_da_foto_do_autor"
}
~~~
